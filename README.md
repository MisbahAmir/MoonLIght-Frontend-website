# 🌙 Moon Light Events

> ✨ A modern and elegant event management website designed to showcase services, packages, and event experiences.

---

## 📌 Description
Moon Light Events is a front-end web project developed using HTML, CSS, and JavaScript. The website is designed to represent an event management company, providing users with information about services, event packages, and contact options through a visually appealing and responsive interface.

---

## 🚀 Features
- 🎉 Event Services Showcase  
- 📦 Event Packages Display  
- 🖼️ Attractive UI Design  
- 📱 Fully Responsive Layout  
- 🧭 Smooth Navigation  
- 📞 Contact Section
